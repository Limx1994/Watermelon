"None" 对象
***********

请注意，Python/C API 中并没有直接公开 "None" 的 "PyTypeObject"。由于
"None" 是一个单例，测试对象标识（在 C 语言中使用 "==" 运算符）就足够了
。出于同样的原因也没有 "PyNone_Check()" 函数。

PyObject *Py_None

   Python "None" 对象，表示缺少实际值。该对象没有任何方法并且属于
   *immortal* 对象。

   在 3.12 版本发生变更: "Py_None" 属于 *immortal* 对象。

Py_RETURN_NONE

   从一个函数返回 "Py_None"。
