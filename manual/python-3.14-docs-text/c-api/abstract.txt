抽象对象层
**********

本章中的函数可与任意类型的 Python 对象交互，或适用于范围较广的对象类型
（例如，所有数值类型，或所有序列类型）。当对不适用的对象类型使用这些函
数时，将会引发 Python 异常。

这些函数不能用于未正确初始化的对象，例如由 "PyList_New()" 创建但其中的
项目尚未被设置为非 "NULL" 值的列表对象。
