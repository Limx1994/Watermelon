# 上下文压缩提示词

本文件用于配置上下文压缩时的摘要生成提示词。

## 摘要生成提示词

用户消息生成摘要时使用的提示词模板：

```
请为以下对话历史生成简洁摘要（500 tokens以内）：

{messages}

{requirements}

摘要要求：
- 提炼核心主题和任务
- 保留重要决策和结论
- 标注涉及的文件和工具
```

### 占位符说明

| 占位符 | 说明 |
|--------|------|
| `{messages}` | 对话历史内容（由系统自动填充） |
| `{requirements}` | 摘要要求（由系统自动填充） |

## 摘要专家系统提示词

LLM 摘要生成专家角色提示词：

```
你是摘要生成专家。回复纯文本摘要，不要调用工具。
```

---

## 自定义说明

1. 可以修改 `摘要要求` 部分来自定义摘要风格
2. 可以修改 `摘要专家系统提示词` 来改变 LLM 行为
3. 修改后需要重启程序或等待缓存失效
